Spark - spark is open-source cluster-computing framework which is built around speed,ease of use and streaming analytics
Python is general purpose high level programming language. It provides wide range of libraries for ML and DS
PySpark - it is a python API fro spark majorly used for DS & analysis , using pyspark you can work with spark RDD 

Basic life cycle of PySpark program - create RDDs,Lazy transformation,cache RDDs, perfomr actions

RDD(resilient distributed dataset) - is the building block of every spark application and is immutable
Transformations in RDDs - map,flatMap,filter,distinct,reduceByKey,mappartitions,sortBy
Actions in Rdds - collect,collectAsMap,reduce,countByKey/countByValue,take,first
